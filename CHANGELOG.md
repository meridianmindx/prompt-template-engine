# Changelog

All notable changes to this project will be documented in this file.

## [1.1.0] - 2026-07-02

### Added
- Improved template descriptions
- Added `generate()` convenience function for quick template usage
- Expanded template collection with 10+ new templates
- Added unit tests for all template categories
- Added script to auto-update templates from the library

### Changed
- Updated README with improved usage examples
- Updated test_run.py with more comprehensive assertions

### Fixed
- Resolved import issues in `__init__.py`

## [1.0.0] - 2026-07-01

### Added
- Initial release of prompt-template-engine
- 20+ curated LLM prompt templates across 5 categories
- Cold outreach, content generation, sales copywriting, technical docs, social media templates
- Programmatic API: `get_templates()`, `get_template()`, `generate()`
- `__main__` entry point for CLI usage
