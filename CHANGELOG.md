# Changelog

All notable changes to this project will be documented in this file.

## [2.0.0] - 2026-07-02

### Changed
- Expanded template library to 27+ templates across 5 categories
- Improved template quality and variety
- Added more detailed descriptions for each template

### Added
- Expanded cold outreach templates with personalized messaging
- Content generation templates for blog posts, social media, and newsletters
- Sales copywriting templates for product pages and sales emails
- Technical documentation templates for API docs and release notes
- Social media templates for Twitter, LinkedIn, and Reddit
- Programmatic API: `get_templates()`, `get_template()`, `generate()`
- `__main__` entry point for CLI usage

## [1.0.0] - 2026-07-01

### Added
- Initial release of prompt-template-engine
- 20+ curated LLM prompt templates across 5 categories
- Cold outreach, content generation, sales copywriting, technical docs, social media templates
- Programmatic API: `get_templates()`, `get_template()`, `generate()`
- `__main__` entry point for CLI usage
